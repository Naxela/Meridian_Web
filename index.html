Version 2.0 Features

🌌

One-Click Export

Export your entire scenes to Godot with a single click. Meridian handles all the heavy lifting, setting up the scene, materials, and lighting for you.

⚡

Preprocessing

A background process that automatically optimizes your meshes and textures directly inside Godot for peak performance.

💡

Lightmapping Workflow

A fully Cycles based lightmapping workflow, which includes encoding and custom shaders supporting lightmap blending.

🔄

LiveLinking

Realtime TCP livelinking from Blender to Godot. Supports mesh, camera, and light syncing.

🖥️

Desktop Pipeline

Full support for a robust desktop pipeline. Export directly to binaries without ever opening the Godot editor.

🥽

XR Pipeline

Set up spatial computing workflows using the integrated XR tools.

✨

Postprocessing

Includes new visual effects and shaders: Tilt Shift, Colorgrading, Chromatic Aberration, Sharpen, Vignette, and more.

🪞

Reflection Probes and decals

Seamless implementation and export support for generating Godot reflection probes and decals directly from your 3D scene.

Exporter Interface

Configure renderer settings and publish platforms. Meridian integrates into your workflow to help automate project setup.
