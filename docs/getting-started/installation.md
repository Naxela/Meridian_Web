# Installation

Follow these steps to install the Meridian addon into both Blender and your Godot project.

## Requirements
* Blender 4.0 or higher
* Godot Engine 4.2 or higher

## Blender Setup

1. Download the `meridian-addon.zip` archive.
2. In Blender, navigate to `Edit > Preferences > Add-ons`.
3. Click **Install...**
4. Locate the downloaded `.zip` and install it.
5. Check the box next to **Pipeline: Meridian 2.0** to enable the addon.

## Godot Setup

1. Copy the `addons/meridian` folder into your Godot project's `res://addons/` directory.
2. Navigate to `Project > Project Settings > Plugins`.
3. Locate **Meridian** and check the **Enable** box.

You're now ready to use livelinking and desktop exports!
