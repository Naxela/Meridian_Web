# Welcome to Meridian 2.0

**Meridian** is the ultimate pipeline toolkit for pushing data and visuals seamlessly from **Blender** directly to the **Godot Engine**. 

Whether you're developing high-end desktop releases, setting up baked XR pipelines, or just managing massive scenes, Meridian automates all the heavy lifting for you.

## Core Features

Our documentation dives deep into every component. Ensure you read up on:

* **Preprocessing:** Optimize your meshes and materials effortlessly.
* **Lightmapping Workflows:** Break the resolution barriers.
